Football Statistics Applet Demo

FA Premier League Statistics

This page provides historical data and statistics for the FA Premier League. By default it shows the league table, results, form guide and graphs for the current season. However, by changing the value of the "Season" drop-down box you can view data from any year. Alternatively, by selecting "(All Time)" from the "Season" drop-down, you can view the all-time Premier League table and statistics.

This page is intended to demonstrate the capabilities of the Football Statistics Applet. The version of the software embedded on this page is a preview build of version 3.0.
