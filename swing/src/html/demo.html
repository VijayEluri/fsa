FA Premier League Statistics

This page provides historical data and statistics for the FA Premier League. By default it shows the league table, results, form guide and graphs for the current season. However, by changing the value of the "Season" drop-down box you can view data from any year. Alternatively, by selecting "(All Time)" from the "Season" drop-down, you can view the all-time Premier League table and statistics. The data files include every match played since the competition's commencement on 15th August 1992.

This page is intended to demonstrate the capabilities of the Football Statistics Applet. The version of the software embedded on this page is a preview build of version 3.0. It is still under active development. If you find any problems with it, or have any suggestions for enhancements please use the issue tracker or e-mail dan at uncommons dot org.
